# user-api
CRUD Operations <br> <br> Echo - Swaggo - Logrus - Mongo  


 ##  Requirements

 - Docker


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/marikann/user-api
   ```
2. Change Folder
   ```
   cd to user-api folder
3. Run Docker Compose
   ```
   docker-compose up
   ```
4. Go this link
   ```
   http://localhost:8080/swagger/index.html
   ```
